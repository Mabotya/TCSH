# TCSC
■ TCSCについて
TCSCは、TRAIN CREWのカスタムサウンド機能をより直感的に扱えるようにするための補助ツールです。

次のような機能を使用することができんす。
・直感的な操作による管理
・指定形式への自動変換
・音声ファイルの秒数指定
・音声のモノラル化
・プリセット保存機能
など

ここを英語に

・対応状況はTRAIN CREW側の仕様変更により変わる場合があります。

■ SmartScreenの警告について
本ソフトは個人開発のWindows向けソフトのため、初回起動時にMicrosoft Defender SmartScreenの警告が表示される場合があります。
これは、ソフトのダウンロード実績が少ない場合や、発行元が不明な場合に表示されることがあります。
実行する場合は、配布元が正しいことを確認したうえで、「詳細情報」→「実行」を選択してください。
不安がある場合は、実行しないでください。
